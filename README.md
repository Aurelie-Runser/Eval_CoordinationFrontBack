# Eval - Coordination Front & Back - Aurélie Runser

Pour récupérer le projet : 
```
git clone https://github.com/Aurelie-Runser/Eval_CoordinationFrontBack
cd Eval_CoordinationFrontBack
```

## Installez les dépendances

### Backend
```
cd backend
mvn clean install
```

### Frontend
```
cd frontend
npm i
```

## Lancez le Backend
```
cd backend
mvn spring-boot:run
```

## Lancez le frontend
```
cd frontend
npm run start
```
